# vanta design.
The delicate UI toolkit, focused on accessibility, performance, and client side rendering application. Includes 30+ components, a lot of tools and [Flowness](https://github.com/tyeongkim/flowness/), A React application flow control library.

> vanta design is an internal project under active development. <br>
> Not yet publicly released. Source is available under [MIT + CC license](https://github.com/vanta-design/desktop/blob/main/LICENSE), except Flowness.

## Highlights
- 30+ popular components
- Native Light / Dark mode
- Customizable color mode
- Built with React
- Client Side Rendering, ESM only

## Principles
- **Accessibility first** <br>
  Every component are designed with semantic structure at its core.
- **Lightweight on runtime** <br>
  Reduced bundle size as much as possible on production environment.
- **Sustainable engineering** <br>
  Prioritize client-side boundaries and predictable execution patterns for long-term maintainability.
